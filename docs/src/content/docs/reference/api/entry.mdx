---
title: Entry Reference
description: A reference for the Entry object.
---

An `Entry` represents a key and value pair, used in [`HashMap`](/reference/api/hashmap)

```ql
var entry = Entry("key", "value");
```

## Fields

### `key`

The key for the entry.

```ql
var entry = Entry("key", "value");
var key = entry.key; // "key"
```

#### Type

`dyn!`

### `value`

The value for the entry.

```ql
var entry = Entry("key", "value");
var value = entry.value; // "value"
```

#### Type

`dyn!`
