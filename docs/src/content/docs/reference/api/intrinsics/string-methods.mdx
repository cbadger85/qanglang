### `char_at`

Returns the string character at the given index.

```ql
var string = "Hello World!";
var second_letter = string.char_at(1); // "e"
```

#### Parameters

index: `Number` - The index of the character. if a negative number or out-of-bounds number is supplied, `nil` will be returned.

### Returns

`String` - The character as a string at the given index.

### `concat`

Returns a string of both values concatenated together.

```ql
var string1 = "Hello ";
var string2 = "World!";
var combined_string = string1.concat(string2) // "Hello World!"
```

#### Parameters

other: `String` - The other string to combine with.

### Returns

`String` - The combined string.

### `contains`

Checks if a string contains a value.

```ql
var string1 = "Hello World!";
var string2 = "llo W";
assert(string1.contains(string2));
```

#### Parameters

term: `String` - The string to search for.

### Returns

`Boolean` - `true` if the string contains the term, `false` otherwise.

### `ends_with`

Checks if a string ends with a value.

```ql
var string1 = "Hello World!";
var string2 = "World!";
assert(string1.ends_with(string2));
```

#### Parameters

term: `String` - The string to search for.

### Returns

`Boolean` - `true` if the end of the string matches the term, `false` otherwise.

### `split`

Returns an array of strings split at the provided delimeter.

```ql
var string = "Hello World!";
var arr1 = string.split(); // ["H", "e", "l", "l", o", " ", "W", "o", "r", "l", "d", "!"]
var arr2 = string.split(" "); // ["Hello", "World!"];
```

#### Parameters

delimter: `String?` <Badge text="optional" /> - The delimeter to use when splitting the arrays into strings. If none is provided, `""` is used.

#### Returns

`String` - An array of strings split by the delimeter.

### `length`

Returns the length of the string.

```ql
var string = "Hello World!";
assert_eq(string.length(), 12);
```

#### Parameters

None

#### Returns

`Number` - The length of the string.

### `starts_with`

Checks if a string starts with a value.

```ql
var string1 = "Hello World!";
var string2 = "Hello";
assert(string1.starts_with(string2));
```

#### Parameters

term: `String` - The string to search for.

### Returns

`Boolean` - `true` if the start of the string matches the term, `false` otherwise.

### `to_lowercase`

Returns a string with all characters lowercased.

```ql
var string = "Hello World!";
var lowercase_string = string.to_lowercase(); // "hello world!"
```

#### Parameters

None

#### Returns

`String` - The lowercased string.

### `to_number`

Returns a [`Result`](/reference/api/result) with either the converted value wrapped in `Ok`, or an `Err`.

```ql
var number_as_string = "1234";
var number = number_as_string.to_number().unwrap(); // 1234

var not_a_number = "x";
assert_throws(() -> not_a_number.to_number().unwrap()); // true
```

#### Parameters

None

#### Returns

`Result` - The result of the conversion. If successful, the number will be wrapped in `Ok`, otherwise an `Err` will be returned.

### `to_uppercase`

Returns a string with all characters uppercased.

```ql
var string = "Hello World!";
var uppercase_string = string.to_uppercase(); // "HELLO WORLD!"
```

#### Parameters

None

#### Returns

`String` - The uppercased string.
