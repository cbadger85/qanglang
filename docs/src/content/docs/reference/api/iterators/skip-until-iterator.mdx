title: SkipUntilIterator Reference
description: A reference for the SkipUntilIterator implementation
---

A `SkipUntilIterator` skips values while the predicate is false. It inherits from [`Iterator`](/reference/api/iterators/iterator/).

## Constructor

Creates an iterator to take items from another iterator.

```ql
var iterator = ArrayIterator([0, 1, 2]);
var skip_iterator = SkipUntilIterator(iterator, (n) -> n < 1);
```

#### Parameters

iterator: `Iterator` - The iterator to skip items from.

predicate: `(dyn?) -> Boolean` - The predicate to stop skipping at.

## Methods

### Inherited

All methods are inherited from the `Iterator` class. See the [Iterator reference](/reference/api/iterators/iterator#methods) for documentation.

## Example

```ql
var iterator = ArrayIterator([0, 1, 2]);
SkipUntilIterator(iterator, iterator, (n) -> n < 1).for_each(print_ln);
// 1
// 2
```
